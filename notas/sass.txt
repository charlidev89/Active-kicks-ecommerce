Sass  
Es un preprocesador de css que nos permite escribir codigo css  de forma mas legible, mantenible  y escalable
Compone de distintas herramientas o funciones como son:
variables, anidación de selectores, funciones (denominadas mixins), 


documentation:
https://sass-lang.com/documentation/

Recurso:
https://desarrolloweb.com/articulos/anidacion-selectores-sass.html


<<<<<<<<<<<<<<<   Conceptos:

sass --version  : permite ver la version de sass 

Vinculacion:
sass --watch origen.scss destino.css  ---->  "todo lo que se escriba en Scss se traduce en CSS"

<<<<<<<<<<<<<<<<<   Variables:
$primary-color:#33223;

<<<<<<<<<<<<   Anidamiento:   permite dar estilo de forma de cascada.

header{
    background-color:2123;

    nav{
        padding:20px;
        ul{

            li{
                text-decoration:underline;

            }
        }
    }
}



<<<<<<<<<<<<<<<<<<<                      extends:
/ Definición de una clase base
.button {
  padding: 10px 20px;
  border-radius: 5px;
  color: white;
  background-color: blue;
}

// Extensión de la clase base en otras clases
.primary-button {
  @extend .button;
  background-color: blue;
}



<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<          mixins :  
 son  una especie de funciones

// Definición de un mixin con parámetros
@mixin button-styles($padding, $border-radius, $bg-color) {
  padding: $padding;
  border-radius: $border-radius;
  color: white;
  background-color: $bg-color;
}

// Uso del mixin en clases
.primary-button {
  @include button-styles(10px 20px, 5px, blue);
}

.secondary-button {
  @include button-styles(10px 20px, 5px, gray);
}




<<<<<<<<<<<<<<<<<<<<<<<<<<<<<< Parcials:
son archivos que pueden tener parte del codigo 

_variables.scss   (archivo que contiene todas las variables del proyecto)
_header.scss (archivo que contiene todo el estilo del header)

// @import  'variables';    "llamar en el archivo main de scss"
// @import 'header';      "llamar en el archivo main de scss"



IMPORTANTE:

Para pasar un archivo css a sasss
1) se usa como guia el css original 
2) debemos  separar todo el css original  en secciones para organizar todo  y sea mas facil pasar a sass
3) crear cada archivo parcial de cada seccion con su correspoondiente division de carpeta  :  " 
scss/
├── main.scss           (archivo principal en la raíz)
├── abstracts/          (variables, mixins, functions)
├── base/               (reset, typography, base elements)
├── components/         (buttons, forms, etc.)
├── layout/             (header, footer, grid)
└── ...
4) el main scss debe tener los imports de cada seccion.
5)  para comenzar a pasar a sass, debemos pegar el codigo css original a cada parcial sasss e ir "traduciendolo" a sass  crear los mixins , variables,  anidar de forma correctaetc.
6) una vez creado todo, debemos vincularlo con el archivo original de css con el :  sass --watch scss/main.scss:css/style.css  , el archivo original css cambiará a lo que traduce el mainscss de sass.
